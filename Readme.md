# Defra flood warning layout

Defra page template for internal services

## How to install

`npm install cathydutton/layout-defra`

## How to include

Replace this line 

`{% extends "layout.html" %}`

With...

`{% extends 'layout-defra.html' %}`

Set the service name... 

`{% set DefraServiceName = "My service name" %}`
